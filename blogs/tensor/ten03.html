搭建神经网络八股

自学神经网络是一个略显枯燥的过程，但是当初学者们走过最初的一点泥泞与坎坷，来到搭建神经网络八股的这一步时，就能领略到神经网络的简洁与美丽。之所以称之为八股，是因为哪怕你是一个完全没有了解过的小白，也能根据介绍搭建出属于自己的神经网络。

用TensorFlow API：tf.keras 搭建神经网络

Sequential六步法：sequential代码提纲顺序网络结构，无法搭建出跳转网络结构

①import引入搭建网络必须的包

②引入训练集和测试集

③搭建网络结构，描述各层神经网络 Model=tf.keras.model.Sequential([网络结构])

网络结构举例：

拉直层：tf .keras.layers.Flatten( )

全连接层：tf .keras.layers.Dense（神经元个数，activation=“激活函数”，kernel_regularizer=正则化类型）

activation（可根据函数名进行配置）：可选relu、softmax、sigmoid、tanh kernel_regularizer：可选tf.keras.regularizer.l1()、tf.keras.regularizer.l2()

L1（拉格朗日）正则假设参数的先验分布是Laplace分布，可以保证模型的稀疏性，也就是某些参数等于0

L2（岭回归）正则假设参数的先验分布是Gaussian分布，可以保证模型的稳定性，也就是参数的值不会太大或太小。

在实际使用中，如果特征是高维稀疏的，则使用L1正则；如果特征是低维稠密的，则使用L2正则。

④配置训练方法 Model.compile配置优化器、损失函数、评价指标。

Model.compile(optimizer=优化器，loss=损失函数，metrics=["准确率"]) optimizer可选：

‘sgd’ 或者 tf.keras.optimizer.SGD (lr=学习率，momentum=动量参数）

‘adagrad’ 或 tf.keras.optimizer.adagrad(lr=学习率)

‘adadelta’ 或 tf.keras.optimizer.Adadelta（lr=学习率）

‘adam’ 或 tf.keras.optimizer.Adam(lr=学习率,beta_1=0.9,beta_2=0.999)

beta指的是一阶和二阶指数衰减率

loss可选：

‘mse’ 或 tf.keras.losses.MeanSquaredError()

‘tf.keras.losses.categorical_crossentropy’和‘tf.keras.losses.sparse_categorical_crossentropy’ 两个交叉熵函数，sparse的含义是：真实的标签值y可以直接传入int型的标签类别，即有sparse时，不需要进行one_hot()转换为独热码这一个步骤，没有的话就需要该转换步骤。

Metrics可选：

‘accuracy’ ：y和y_都是数值，如y_=[1],y=[1]

‘categorical_accuracy’：y与y_都是独热码（概率分布），如y_=[0,1,0] y=[0.25,0.69,0.06]

‘‘sparse_categorical_accuracy’’：y或者y_中有一个是独热码形式

⑤Model.fit 执行训练过程，告知训练集与测试集的标签和特征，batch的大小 Model.fit(训练集的输入特征，训练集的标签，batch_size= ,epochs= ，validation_data=(测试集的输入特征，测试集标签)，validation_split= 从训练接划分百分之几给测试集， validation_freq=多少次epoch测试一次)

⑥Model.Summary 打印网络结构和参数统计

以下部分是用sequential结构搭建神经网络对服装进行识别的代码，建议电脑观看阅读。可直接复制粘贴使用，修改参数去体验不同参数之间的区别：


import tensorflow as tf
# ①引入tensorflow包
fashion = tf.keras.datasets.fashion_mnist
# ②从Keras库中的数据库中引入需要的数据
(x_train, y_train), (x_test, y_test) = fashion.load_data()
# 按格式加载数据
x_train, x_test = x_train / 255.0, x_test / 255.0
#使0-255之间的图片灰度值变为0-1之间的数值，更适合神经网络吸收
# ③配置sequential顺序结构搭建网络
model = tf.keras.models.Sequential([
    tf.keras.layers.Flatten(),
    #拉直层，使图片展开成一维数组
    tf.keras.layers.Dense(128, activation='relu'),
    #全连接层，128个神经元，激活函数是relu
    tf.keras.layers.Dense(10, activation='softmax')
    #输出层，10个神经元（输出神经元的个数是由最终的可能结果确定的）
])
# 至此，网络结构搭建完毕，如果需要添加层数只需要在sequential结构中添加即可
# ④训练及优化
model.compile(optimizer='RMSprop',
              #优化方法：可以根据需要选择不同的优化方法，不同方法的优缺点在之前的笔记中有写
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=False),
              #交叉熵损失函数，如果计算的结果为数值，此处为true。计算结果为概率，此处为false
              metrics=['sparse_categorical_accuracy'])
              #根据y与y-的数据类型，选取不同的方法计算准确率
# ⑤执行训练过程
model.fit(x_train, y_train, batch_size=32, epochs=6, validation_data=(x_test, y_test), validation_freq=1)
# ⑥输出结果
model.summary()

Class六步法：相较于sequential，class可以跳转。二者的整体结构区别并不大，只在网络搭建部分有较大区别。

import tensorflow as tf
# 引入包
from tensorflow.keras.layers import Dense, Flatten
# 引入网络结构层
from tensorflow.keras import Model
# 引入Model函数
fashion = tf.keras.datasets.fashion_mnist
# 导入数据集
(x_train, y_train), (x_test, y_test) = fashion.load_data()
x_train, x_test = x_train / 255.0, x_test / 255.0
# 数据集处理，使神经网络更容易接受
# 定义类。搭建网络结构
class MnistModel(Model):
    def __init__(self):
        super(MnistModel, self).__init__()
        self.flatten = Flatten()
        self.d1 = Dense(128, activation='relu')
        self.d2 = Dense(10, activation='softmax')

    def call(self, x):
        x = self.flatten(x)
        x = self.d1(x)
        y = self.d2(x)
        return y
    pass
# init定义model网络结构的基础信息，CALL定义搭建的网络结构
# model执行调用
model = MnistModel()
# 配置优化
model.compile(optimizer='Adagrad',
              loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=False),
              metrics=['sparse_categorical_accuracy'])
# 执行训练
model.fit(x_train, y_train, batch_size=32, epochs=4, validation_data=(x_test, y_test), validation_freq=1)
# 输出结果
model.summary()
